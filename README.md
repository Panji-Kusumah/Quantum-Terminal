## 📦 Installation

Clone the repository.

```bash
git clone <repository-url>
```

Navigate to the project directory.

```bash
cd <project-folder>
```

Install dependencies.

```bash
npm install
```

Start the frontend.

```bash
npm run dev
```

Start the backend.

```bash
npm run server
```
